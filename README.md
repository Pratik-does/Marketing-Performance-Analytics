# Marketing Performance Analytics Dashboard

An end-to-end marketing analytics project that combines **Power BI**, **SQL**, **Python**, and **Streamlit** to analyze multi-channel campaign performance across platforms, countries, objectives, and time periods.

This project was built to answer one question:

> **Where is marketing spend creating the most value, and what should be optimized next?**

---

## Project Overview

This portfolio project presents a complete business intelligence workflow for marketing performance analysis.

It includes:
- data cleaning and preparation
- SQL-based analysis
- Power BI dashboard development
- executive insights and recommendations
- presentation-ready reporting
- Streamlit deployment for portfolio showcase

The dashboard focuses on:
- total spend
- total revenue
- profit
- ROI
- clicks
- conversion rate
- platform performance
- country performance
- campaign performance trends

---

## Business Questions Answered

This project is designed to help answer:

- Which platforms generate the highest revenue?
- Which channels are the most efficient for conversion?
- How does spend compare with revenue over time?
- Which countries contribute the most to marketing spend?
- Which campaigns are performing best?
- Where should budget be reallocated?
- What are the main improvement opportunities?
