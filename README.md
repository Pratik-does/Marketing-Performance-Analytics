# Marketing Performance Analytics Dashboard

A professional marketing analytics project built to analyze multi-channel campaign performance, customer engagement, revenue contribution, and marketing efficiency across platforms and countries.

## Project Summary

This project combines:
- Power BI dashboarding
- Business insights analysis
- Monthly performance reporting
- Campaign-level review
- Streamlit-based portfolio presentation

## Business Goals

The project answers questions such as:
- Which platforms generate the highest revenue?
- Which channels deliver the strongest conversion performance?
- How does spend compare with revenue over time?
- Which countries contribute the most to marketing spend?
- Which campaigns and objectives perform best?

## Key Deliverables

- Interactive Power BI dashboard
- Business insights and recommendations report
- Presentation deck
- Streamlit portfolio app
- Supporting SQL, notebook, and data files
